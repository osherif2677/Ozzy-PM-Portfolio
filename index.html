Excel ¬∑ Store-level KPI control sheet

Monthly tracker combining Inventory Health, INF / SDU, shrink and Demand vs Actuals. Uses pivots, INDEX/MATCH and conditional formatting to highlight risk. 

// Sample structure (one row per month & department)

   0))

// Example: flag high risk months (INF > 10% or Shrink > 15%)
=IF(OR([@[INF_Pct]]>0.10,[@[Shrink_Pct]]>0.15),"High Risk","OK")
SQL style ¬∑ INF / SDU aggregation

How I‚Äôd translate my Excel logic into SQL once I have warehouse access. 

-- INF and SDU by department & month
SELECT
  DATE_TRUNC('month', order_date) AS month,
  department,
  SUM(CASE WHEN outcome = 'INF' THEN 1 ELSE 0 END)::float /
  COUNT(*)                         AS inf_pct,
  SUM(CASE WHEN outcome = 'SDU' THEN 1 ELSE 0 END)::float /
  COUNT(*)                         AS sdu_pct
FROM outbound_orders
GROUP BY 1, 2
ORDER BY 1, 2;
