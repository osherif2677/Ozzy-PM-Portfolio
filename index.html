How I Work with Data & Tools

Excel · SQL · Python · R

I don’t just read dashboards. I pull data into my own trackers, clean and model it in Excel / SQL / Python / R, then turn it into clear mechanisms for my team.

Tools & Methods I Use Daily
  • Excel Pivot tables, INDEX/MATCH & XLOOKUP, nested IF / IFS, conditional formatting, dynamic ranges, and slicers to track Inventory Health KPIs, shrink buckets, and Outbound INF/SDU by month.
  • SQL Querying INF & SDU slices, pulling demand vs actuals, and joining reports by ASIN, department, and day to support deeper analysis.
  • Python Simple Pandas notebooks to experiment with trend detection (e.g., rolling averages, moving INF spikes) and to automate some data reshaping.
  • R Exploring time-series visualizations and basic statistical checks when I want to validate whether changes are real or just noise.
  • Smartsheet & Sheets Lightweight trackers for project timelines, RAID logs, and cross-team updates when partnering with other leaders.
